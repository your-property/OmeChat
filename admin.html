Quick Guide

Banning users: Go to Users tab → click Ban next to any user, or use the Actions tab to ban by email.

Unbanning: When a user pays the $5 unban fee and submits their TX ID, verify payment in Payoneer → then unban from the Users tab.

Activating Premium: When a user pays the $25 premium fee, verify payment → then activate from the Users tab or Actions tab.

Reports: View flagged sessions in Reports tab. Action = resolve with possible ban. Dismiss = false alarm.

Export: Use Export Data tab to download login_details.json and ban_list.json files.
